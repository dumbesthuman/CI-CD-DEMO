# 🚀 CI/CD Demo using GitHub Actions (Python)

## 📌 Overview
This project demonstrates a simple **CI/CD pipeline** using GitHub Actions.  
It automatically runs tests on every code push and simulates deployment.

---

## ⚙️ What is CI/CD?

- **CI (Continuous Integration):** Automatically tests code when changes are pushed.
- **CD (Continuous Deployment):** Automatically deploys code after successful tests.
